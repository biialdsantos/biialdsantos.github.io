esad events app

As part of an academic project, I developed a prototype for a mobile application dedicated to promoting and organizing events at ESAD.CR. The app aims to centralize all event-related information — including exhibitions, talks, and workshops — in one accessible and user-friendly platform for students, faculty, and visitors.

main task

The project focused on intuitive navigation, clear visual hierarchy, and a modern UI that reflects the school's creative identity. Key features include an interactive calendar, event registration, push notifications, and visual highlights of past and upcoming events.
